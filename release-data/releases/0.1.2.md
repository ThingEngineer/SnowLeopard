# SnowLeopard 0.1.2

## Highlights

- Extends the history page with a longer 72-hour range option.
- Clarifies history gap labeling so restart boundaries are easier to understand.
- Improves update-page visibility and status behavior when an update is available.

## Release notes

This release makes the device easier to read and operate day to day. The biggest visible changes are on the history page and the update page.

### What changed from 0.1.1

- The history page now includes a 72-hour range button.
- History gap labeling is clearer when the chart crosses a restart boundary.
- The update action appears only when an update is actually available.

## Update notes

- Open the History page to use the new 72-hour view.
- Use the Settings page to check for updates when you want the latest firmware.
- After upgrading to 0.1.2, the History chart may start empty until new samples are collected.
- If you compare history across reboots, the gap labels are now easier to interpret.
