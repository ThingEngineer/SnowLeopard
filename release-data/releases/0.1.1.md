# SnowLeopard 0.1.1

## Highlights

- First stable update release for everyday use.
- Improves update reliability with stronger file verification checks.
- Shows clearer update progress and status on the device.
- Publishes full release notes and download links from the public portal.

## Release notes

This release keeps the same core SnowLeopard features and focuses on making updates safer and easier to follow.

### What changed from 0.1.0-dev

- Added a published update file for direct firmware installs.
- Added stronger trust and checksum checks during update validation.
- Improved Settings page update messages so progress is easier to understand.
- Finalized the public release pages for notes and download links.

## Update notes

- If you are on 0.1.0-dev, this is the recommended next update.
- Use Check for update on the Settings page before starting install.
- Keep power and Wi-Fi stable until the device finishes updating and restarts.
