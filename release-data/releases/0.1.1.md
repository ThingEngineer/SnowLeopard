# SnowLeopard 0.1.1

## Highlights

- Publishes the first real GitHub-hosted firmware release for SnowLeopard.
- Serves release notes and OTA metadata from the repository's GitHub Pages site.
- Adds stricter OTA protections with pinned GitHub trust roots and SHA-256 verification.
- Improves on-device Firmware Update feedback with progress and verification status.

## Release notes

This is the first public release intended for real OTA distribution. Devices can check the GitHub-hosted manifest, compare their installed version against the latest published release, and show clearer progress while downloading and validating an update.

## Operator notes

- After the tagged release is built, replace the placeholder `sha256` and `size` values in `release-data/current.json` with the real artifact metadata.
- Confirm GitHub Pages is enabled in repository settings with source set to GitHub Actions.
- Test `/settings` on a device running `0.1.0-dev` and confirm that `0.1.1` appears as the latest version.
