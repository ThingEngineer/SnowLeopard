# SnowLeopard 0.1.0-dev

## Highlights

- First full SnowLeopard release with setup, cooling control, and web access.
- Adds status, settings, and history pages for everyday monitoring and tuning.
- Includes alarms, display options, and optional settings-password protection.
- Introduces update checks and hosted release notes through the public release portal.

## Release notes

This release is the first complete SnowLeopard baseline. It brings together device setup, cooling control, monitoring, and update support in one version.

### Included in this baseline

- Wi-Fi setup flow with fallback behavior and access at snowleopard.local.
- Cooling control with Auto, Manual On, and Manual Off modes.
- Adjustable target temperature, thresholds, and compressor timing controls.
- Internal and external temperature readings with calibration adjustments.
- Audible temperature alarms with a built-in test action.
- OLED status display with multiple layout options.
- Web pages for status, settings, and history.
- Optional settings password with on-device reset gesture.
- Saved history that remains useful across restarts.
- Update check support and queued install flow tied to hosted release data.

## Update notes

- If you are running older firmware, review the settings and alarm sections after updating.
- Keep the device powered and connected to Wi-Fi while updates are in progress.
- Use the release notes link in Settings to confirm what changed before updating.
