# SnowLeopard 0.1.0-dev

## Highlights

- Adds an OTA-capable dual-slot ESP32-C3 partition layout.
- Adds the `/settings` Firmware Update section on the device.
- Adds firmware manifest check endpoints and a queued OTA update flow.
- Adds the single-repo GitHub Pages release portal and file-based release metadata.

## Release notes

This is the first release candidate for the GitHub-hosted firmware update workflow. The firmware can now compare its installed version against a manifest in this repository and surface update availability in Settings.

## Operator notes

- Before publishing a real firmware release, update `include/firmware_version.h` and `release-data/current.json` together.
- Replace the placeholder GitHub Release binary URL with the real release asset URL.
- Fill in the `sha256` and `size` fields once the release binary is built.
