# SnowLeopard 1.0.0

## Highlights

- Adds button control setting to lock physical buttons when needed.
- Reduces alarm beep frequency for quieter alerting.
- Streamlines the root page display for better visibility.
- Improves web interface with cleaner notifications and feedback.

## Release notes

This is the first stable release of SnowLeopard. The device has been tested thoroughly and is ready for everyday temperature control and monitoring. This version adds user-requested control options and refines the alarm and display behavior based on real-world use.

### What changed from 0.1.2

- **Button Control**: You can now disable the physical buttons from the Settings page. When disabled, the buttons will not change the setpoint, preventing accidental adjustments. Changes can still be made through the web interface.

- **Quieter Alarms**: The alarm beep interval is now 3 seconds instead of 1 second, making temperature alerts less intrusive while still being noticeable.

- **Cleaner Root Page**: The relay mode indicator has been removed from the main status page to reduce clutter and emphasize temperature readings.

- **Better Web Feedback**: The settings page and other web views now show clearer toast notifications when actions succeed or fail.

## Update notes

- After updating to 1.0.0, visit the Settings page to configure the new Button Control option if you want to lock the physical buttons.
- The alarm beeping pattern will be noticeably quieter (longer pauses between beeps) if a temperature alarm is active.
- The root status page will no longer show the relay mode, focusing on temperature and relay state only.
- All existing settings and history are preserved during the update.
